A modern luxury perfume e-commerce platform built using the MERN stack, featuring a glassmorphic UI, secure authentication, shopping cart, and product reviews.


# Features

~ Luxury Glassmorphic UI with smooth Framer Motion animations

~ Product Catalog with category filters and highlights

~ Shopping Cart with real-time updates

~ User Authentication (JWT-based)

~ Product Reviews & Ratings with helpful voting

~ Wishlist & Social Sharing

~ Responsive Design for mobile, tablet, and desktop


# Tech Stack

<!-- Frontend -->
~React 18, Vite
~Tailwind CSS
~Framer Motion
~React Router

<!-- Backend -->
~Node.js, Express
~MongoDB, Mongoose
~JWT Authentication
~RESTful APIs

# Set up and installation 

<!-- Backend  -->
cd backend
npm install 
node seedData.js (only one time)
node server.js 

<!-- Frontend -->
cd frontend
npm install
npm run dev

# Authentication
~ JWT-based authentication
~ Secure password hashing
~ Protected routes for cart and reviews
